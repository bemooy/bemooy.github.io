关于 我

梦想从来都不是一件浪漫的的事，还是当下最不缺的东西。生在小城，求学省会，打拼在大都会的人，除了年轻的身体什么都没有，更没有资格和城市讨价还价，因为城市永远年轻，而你终将老去。

足球

恒大和巴萨的比赛0:3落幕之后,作为一个巴萨的伪球迷,我也不知道说什么.国足走到这已经不容易了.我们学校开学也有一场足球赛,我做为重量级的特邀选手,四分之一半决赛的时候我上场了,为了给新人机会,我踢的后卫.最后以碾压的优势取胜.但后来不知怎么了,他们不让我上场了,也许是应为我太强势了吧。也应为我的退场,剩下的比赛我们学院磕磕绊绊,不过好歹我们得了冠军！

Java

学习Java已经有一年了,从最初的摸索到现在已经入门,一路走来个中酸爽也只有自己知道.大学带给我们的不只是独立的人格,还有让我们赖以生存的技能.第一次敲出“Hello World”,心中蛮高兴的.一点一点从J2SE到J2EE,在到Struts2 Hibernate Spring Jquery,一直在前进.感谢那些一直在指导的老师
